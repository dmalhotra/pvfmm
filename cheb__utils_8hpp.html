Computes Chebyshev approximation from function values at cheb node points.

Parameters
[in]fn_vThe array of function values on a tensor product Chebyshev grid (first kind nodes) of size dof*(deg+1)^3.
[in]degThe degree of the Chebyshev approximation.
[in]dofThe number of real scalar values associated with each Chebyshev node point.
[out]cheb_coeffThe coefficient array of size dof*(deg+1)*(deg+2)*(deg+3)/6.
Returns
Estimate of the truncation error.
